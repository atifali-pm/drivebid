# Screenshots

Drop PNG screenshots into this folder. The root README references these filenames:

| Filename | What it shows |
|---|---|
| `01-login.png` | Login / register screen |
| `02-rider-map-picker.png` | Rider picking pickup + dropoff on the map (search + click) |
| `03-rider-live-bids.png` | Live bid list with driver name, price, ETA |
| `04-driver-open-rides.png` | Driver browsing open ride requests with mini-maps |
| `05-trip-in-progress.png` | Active trip card with Start / Complete buttons |
| `06-rating.png` | Post-trip 5★ rating form |

Recommended: 1440×900 viewport, light theme, no personal browser extensions visible in chrome.

After adding files, uncomment the image references in the root [README](../../README.md) screenshots section (or just rewrite that section to include them inline).
